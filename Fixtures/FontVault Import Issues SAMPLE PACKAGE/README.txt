FontVault Import Issues — Review Package (Sample)

This is a pre-built example of what FontVault creates via
Import Details → Export Review Package…

Contents:
  Failed/              — sample copies of failed imports
  Naming/              — sample copies of naming-review imports
  Import Issues.html   — report (links target files in this folder)
  README.txt           — this file

Originals are not modified. In a real export, files are copied from
your import source paths.

Open Import Issues.html in Safari to browse.
